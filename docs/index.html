CUI Class

CUI class adalah sebuah library yang digunakan untuk melakukan mengelola view agar lebih teratur dan mengurangi pembuatan view yang berulang-ulang

Fitur:

  • Merender view otomatis sesuai dengan class dan Function
  • Merender view secara parsial untuk beberapa section di halaman
  • Pemilihan layout dalam pemanggilan view
  • Header content yang dapat diubah dalam parameter di Controller
  • Pemanggilan file javascript dan CSS yang terstruktur
  • Mengubah view kedalam berkas Document (.doc) dan Excel (.xls)

Parameter yang pada Function render:

$this->cui->render($data, 'nama layout', $header, 'nama view')

Menampilkan View dasar

Fungsi ini digunakan untuk melakukan render view pada halaman.

$this->cui->render();

View yang dipanggil adalah view yang berada dalam folder yang bernama sama dengan Controller, dan file yang bernama sama dengan Function.

Contoh: Controller bernama blog, Function bernama detail, maka view yang dipanggil secara otomatis adalah views/blog/detail.php

Menampilkan View dengan parsing nilai

Fungsi ini digunakan untuk melakukan render view pada halaman dengan nilai.

Contoh pada Controller: $data['judul'] = "Posting Pertama";
$data['isi'] = "Ini merupakan posting pertama";
$this->cui->render($data); Fungsi ini sama dengan penggunaan akan menampilkan view yang berada dalam folder views/__nama controller__/__nama function__ dengan menggunakan layout default.

Menampilkan View dengan layout

Fungsi ini digunakan untuk melakukan render view pada halaman dengan layout yang telah ditentukan.

Contoh pada Controller: $data['judul'] = "Posting Pertama";
$data['isi'] = "Ini merupakan posting pertama";
$this->cui->render($data, '1column');

Untuk mengubah layout atau menambahkan layout, ada di folder views/layouts

Secara default layout yang digunakan adalah 1column, namun anda dapat mengubah default layout pada variabel $default_layout di class cui.php
Penamaan layout sama dengan nama file namun tanpa ekstensi .php
Pilihan layout yang ada secara dasar ada:

  • 1 Kolom, '1column'
  • 2 Kolom, '2column'
  • 3 Kolom, '3column'

Jika parameter diisi null maka layout default yang akan dipanggil.

$data['judul'] = "Posting Pertama";
$data['isi'] = "Ini merupakan posting pertama";
$this->cui->render($data, null);

Menampilkan View dengan konten header

Fungsi ini digunakan untuk render view dengan konten pada header yang dapat diatur dalam Controller. Jika ada konten header yang tidak didefinisikan namun diharuskan untuk ditampilkan, maka nilai default yang akan digunakan. Nilai default dapat diganti pada variabel-variabel default yang ada pada class cui.php

konten yang dapat diatur secara default antara lain:

Konten Variabel dalam Array Variabel default
Title 'title' $default_title
Author 'author' $default_author
Keyword 'keyword' $default_keyword
Description 'description' $default_description

Contoh dalam Controller:

$header['description'] = "Ini Title";
$header['keywords'] = "cui, tag";
$header['author'] = "Author";
$header['title'] = "Ini Title";
$this->cui->render(null, '1column', $header);

Isi pada file layout agar dapat menampilkan konten header:

Peletakan konten header dapat dilihat pada layout yang bersangkutan (dalam contoh adalah layout 1column, buka folder views/layouts/1column.php untuk melihat lebih detail). Anda dapat menambahkan konten header sendiri sesuai kebutuhan.

Menampilkan View yang berbeda dengan Controller dan Function

Fungsi ini digunakan untuk menampilkan view yang tidak letak folder dan nama file-nya tidak sama dengan nama Controller dan Function-nya, namun tetap ingin mem-parsing nilai dan konten header serta tetap menggunakan layout yang ada.

Contoh pada Controller dengan nama welcome dan Function index

$header['description'] = "Ini Title";
$header['keywords'] = "cui, tag";
$header['author'] = "Author";
$header['title'] = "Ini Title";
$this->cui->render(null, '1column', $header, welcome/viewdata);

Contoh tersebut akan menampilkan view yang berada pada views/welcome/viewdata.php dengan layout 1 kolom.

Menampilkan View secara partial

Fungsi ini digunakan untuk menampilkan view secara partial atau tanpa layout dan dapat dipanggil dalam view juga.

Contoh dalam file layout 1column.php:

Pada contoh diatas, dipanggil view navigation sehingga pada baris tersebut akan menampilkan view pada lokasi views/partial/navigation.php.
Fungsi ini berguna untuk membuat sebuah section pada suatu halaman yang harus dipanggil berkali-kali, sehingga anda cukup membuatnya satu kali dan dapat memanggilnya berkali-kali dengan fungsi ini.

Memanggil JavaScript dan CSS

Struktur dasar

$this->cui->css('nama file CSS tanpa ekstensi');
$this->cui->js('nama file JavaScript tanpa ekstensi');

Untuk file CSS harus berada dalam folder bootstrap/css

Sedangkan file JavaScript harus berada dalam folder bootstrap/js

Contoh penggunaan pada View:

Pada contoh diatas maka CSS yang dipanggil berada dalam lokasi bootstrap/css/bootstrap.css

Sedangkan untuk JavaScript yang dipanggil berada dalam lokasi bootstrap/js/bootstrap.min.js

Anda juga dapat meletakan dalam sub-folder pada tiap folder tersebut, namun sertakan nama folder pada parameter, contoh $this->cui->css('home/style'); untuk file yang berada dalam bootstrap/css/home/style.css

$this->cui->js('home/script'); untuk file yang berada dalam bootstrap/js/home/script.js

Membuat Halaman menjadi PDF

Struktur dasar

$this->cui->to_pdf('nama view', 'data dalam array', 'nama file tanpa extensi');

Contoh pada Controller

View yang di panggil adalah /views/output/report.php

Contoh pada View:

Maka setelah Controller di panggil, view tersebut akan dibuat menjadi berkas PDF dengan nama berkas report.pdf
